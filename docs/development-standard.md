# Development Standard

**Versão:** 1.0

> Documento Mestre dos padrões de desenvolvimento da Jireh Fábrica de Software.

---

# Objetivo

O **Development Standard** é o documento mestre que centraliza todos os padrões técnicos da Jireh.

Ele não descreve regras detalhadas.

Seu objetivo é servir como ponto de entrada para toda a documentação técnica da empresa.

Cada padrão será documentado em um arquivo específico dentro da pasta `docs/standards`.

---

# Princípios

A documentação da Jireh segue os seguintes princípios:

* Um assunto por documento.
* Um documento por responsabilidade.
* Documentos pequenos são mais fáceis de manter.
* Todo padrão deve ser versionado.
* Todo padrão deve possuir sua própria Issue.

# Processo de Evolução

Nenhum padrão deverá ser documentado diretamente neste arquivo.

Cada padrão deverá possuir:

* Documento próprio.
* Issue própria.
* Histórico de alterações.
* Versionamento.
